# SPEC.md - Design Specification

## AI Avatar Studio

### Overview
Dark premium mobile app for AI avatar generation.

### Color Palette
| Name | Hex | Usage |
|------|-----|-------|
| Background | #0D0D0D | Screen backgrounds |
| Surface | #1E1E2E | Cards, dialogs |
| Primary | #6C5CE7 | Buttons, highlights |
| Secondary | #A29BFE | Secondary elements |
| Accent | #FD79A5 | Premium, special |
| Premium Gold | #F9CA24 | Premium badges |

### Typography
- Headings: SpaceGrotesk Bold
- Body: Inter Regular
- Scale: 12, 14, 16, 20, 24, 32

### Spacing (8pt Grid)
- xs: 8px
- md: 16px
- lg: 24px
- xl: 32px

### Border Radius
- Buttons: 16px
- Cards: 16px
- Inputs: 12px
- Pills: 24px

### Screen Flow
1. Splash (2s)
2. Onboarding (3 slides)
3. Home (avatar grid)
4. Photo Selection
5. Style Picker
6. Generation
7. Results
8. Share
